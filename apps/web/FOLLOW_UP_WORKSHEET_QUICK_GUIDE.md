# 覆診記錄工作紙 - 快速使用指南

## 📋 功能簡介

基於勾選的覆診記錄生成列印工作紙，橫向 A4 表格格式。

## 🎯 使用步驟

1. **進入覆診管理頁面**
2. **勾選要匯出的記錄**
   - 在表格左側勾選框選擇記錄
   - 可以使用篩選和搜索縮小範圍
3. **點擊「覆診記錄工作紙」按鈕**
   - 按鈕在勾選記錄後出現在頂部
4. **列印**
   - 系統自動開啟新窗口
   - 使用 Ctrl+P 或 Cmd+P 列印

## 📊 工作紙特點

- **格式**：橫向 A4
- **佈局**：表格形式，一個覆診佔一行
- **排序**：按日期由近到遠自動排序
- **分頁**：每頁約 25 條記錄，自動分頁
- **數量**：無限制

## 📝 包含欄位

日期 | 床號 | 院友姓名 | 出發時間 | 覆診時間 | 覆診地點 | 覆診專科 | 陪診 | 交通 | 備註

## ⚠️ 注意事項

- 必須先勾選至少一條記錄
- 排序固定為按日期由近到遠
- 建議使用 Chrome 或 Edge 瀏覽器列印
- 列印前建議先預覽

## 🔄 與其他匯出的區別

- **工作紙**：列印用，無數量限制，按日期排序
- **記錄表（Excel）**：電子存檔，最多 17 筆，包含覆診袋封面
- **CSV**：數據交換，無限制
